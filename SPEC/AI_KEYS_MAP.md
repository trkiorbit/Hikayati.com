# AI KEYS MAP

## الهدف
توضيح توجيه المفاتيح والمزوّدين داخل المشروع

---

## TEXT
- المسؤول: Story Engine / Text Generation
- المزوّد: مزود النص الحالي الأرخص أو المربوط فعليًا
- الاستخدام:
  - توليد القصة
  - تقسيم المشاهد
  - وصف المشهد الأساسي

---

## IMAGE
- المسؤول: Image Service
- المزوّد: مزود الصور الحالي
- الاستخدام:
  - توليد صور المشاهد
- المدخل:
  - image_prompt النهائي من Story Engine أو Prompt Builder

---

## VOICE NORMAL
- المسؤول: Narration Service
- المزوّد: المزوّد الأرخص
- الاستخدام:
  - قراءة القصة العادية

---

## VOICE CLONE
- المسؤول: Voice Clone Service
- المزوّد: ElevenLabs
- الاستخدام:
  - إنشاء الصوت المستنسخ
  - تشغيل القصة بصوت مستنسخ

---

## VISION / AVATAR
- المسؤول: Avatar Vision Service
- المزوّد: Vision API
- الاستخدام:
  - تحليل صورة الطفل
  - استخراج الملامح
  - استخراج الملابس

---

## القاعدة
- لا تخلط بين المفاتيح
- لا تستخدم ElevenLabs للصوت العادي
- لا تستخدم Image API لتحليل الأفاتار
- لا تستخدم Text API بدل Vision في تحليل الصورة
