# USER JOURNEY

## الهدف
توضيح رحلة العميل من أول دخول حتى العودة والشراء

---

## الرحلة الأساسية
تسجيل الدخول
→ الصفحة الرئيسية
→ إنشاء أفاتار (اختياري)
→ إنشاء قصة
→ مشاهدة القصة
→ حفظها
→ فتحها من مكتبتي لاحقًا

---

## الرحلة مع أفاتار
Login
→ Home
→ اصنع بطلك
→ حفظ الأفاتار
→ Story Wizard
→ تفعيل استخدام الأفاتار
→ Generate
→ Cinema
→ Save
→ My Library

---

## الرحلة مع صوت مستنسخ
Login
→ Home
→ استنساخ صوتك
→ حفظ الصوت
→ Story Wizard
→ تفعيل الصوت
→ Generate
→ Cinema

---

## الرحلة التجارية
Login
→ Home
→ توليد قصة
→ خصم كريدت
→ حفظ
→ العودة لاحقًا
→ مكتبة عامة
→ فتح قصة مدفوعة
→ المتجر
→ شراء كريدت

---

## النتيجة المطلوبة
- المستخدم يفهم ماذا يفعل
- لا يضيع بين الشاشات
- كل خطوة لها هدف واضح
- كل خصم واضح
- كل أصل منشأ له استخدام حقيقي لاحقًا
