### 4. Backend Layer (Supabase - Enhanced)

**Core Tables Schema**:
```sql
-- profiles table
user_id UUID (Primary Key)
name TEXT
credits INTEGER DEFAULT 100
language VARCHAR(10) DEFAULT 'ar'
avatar_data JSONB
voice_clone_enabled BOOLEAN DEFAULT false
created_at TIMESTAMP

-- stories table  
id UUID (Primary Key)
user_id UUID (Foreign Key)
title TEXT
language VARCHAR(10)
scenes_json JSONB
cover_image TEXT
is_public BOOLEAN DEFAULT false
created_at TIMESTAMP

-- public_stories table
id UUID (Primary Key)
title TEXT
summary TEXT
cover TEXT
price_credits INTEGER DEFAULT 10
category VARCHAR(50)
created_at TIMESTAMP

-- purchases table
user_id UUID
story_id UUID
unlock_type VARCHAR(20) -- 'creation', 'access', 'premium'
credits_paid INTEGER
created_at TIMESTAMP
```

**Critical Functions**:
- `deduct_credits(user_id, amount, reason)` - Secure credit management
- Row Level Security (RLS) on all tables
- Storage buckets for images/audio with proper access controls

## PRIVACY & COMPLIANCE FRAMEWORK

### Child Safety Protocols
**Data Collection Restrictions**:
- No permanent storage of children's photos
- No voice cloning for minors
- Parental consent required for voice cloning
- All personal data encrypted at rest and in transit

**Privacy Policy Requirements**:
- Clear explanation of AI usage
- Data retention policies (auto-delete unused content)
- Third-party service disclosures (Pollinations, ElevenLabs, Gemini)
- User rights (data deletion, export, modification)
