# مواصفات تصميم الإطلاق — حكواتي v1

> المرجع البصري: `الهوية البصرية/hakawati_app_visual_identity_and_ideas.md` + `hakawati_logo_v1.png`.
> المعاينة المرئية: `design/launch_ui_v1.html`.

## 1) نظام الألوان (مُعتمد)
- Deep Purple: `#2D0B5A` — الخلفية الأساسية
- Deep Purple 2: `#1A0640` — سطوح ثانوية
- Deep Purple 3: `#0E0326` — قاع التدرج
- Vibrant Orange: `#FF8C00` — الأزرار الرئيسية و CTA
- Orange 2: `#FF6A00` — حواف الأزرار
- Gold: `#FFD700` — الكريدت والحواف المميزة
- Gold 2: `#FFB300` — ظلال ذهبية
- White Warm: `#FFF7ED` — النصوص الرئيسية
- Muted: `#B9A7D9` — النصوص الثانوية
- Glass: `rgba(255,255,255,0.06)` — البطاقات
- Glass Border: `rgba(255,215,0,0.18)` — حواف زجاجية

## 2) الخطوط
- العنوان: Cairo 900 / 700
- النص: Cairo 600 / 500 / 400
- الرقم: Cairo 800 tabular

## 3) المسافات والزوايا
- نصف قطر البطاقة: 18–26
- نصف قطر الأزرار: 999 (دائري) للـ CTA، 18 للبطاقات
- padding داخلي للبطاقة: 14
- المسافة بين الأقسام: 20

## 4) المكونات الموحدة
- **Top Bar**: شعار + اسم التطبيق (يسار RTL) + كريدت (ذهبي) + زر قائمة.
- **Hero Banner**: تدرج بنفسجي↔برتقالي، وهج ذهبي، CTA "أنشئ قصة".
- **Quick Card**: أيقونة متوهجة + عنوان + وصف صغير.
- **Tile (قصة)**: تدرج بنفسجي، وهج برتقالي علوي، عنوان + ميتا. المقفلة تحمل 🔒 وتعرض "افتح بـ ١٠".
- **Bottom Nav (5 عناصر)**: الرئيسية · مكتبتي · أنشئ · المتجر · الإعدادات.
- **CTA**: متدرج برتقالي→برتقالي غامق، نص داكن، ظل ٤٠ px.
- **Credit Pill**: خلفية ذهبية شفافة، حد ذهبي، نص ذهبي.

## 5) الشاشات الأربعة
### الرئيسية
Top Bar → Hero Banner → Quick Actions (اصنع بطلك / اروِ بصوتك) → مكتبتي (أفقي) → المكتبة العامة (أفقي) → Bottom Nav.

### أنشئ قصة
Header بسهم رجوع + كريدت → Stepper (٤ خطوات) → اختر بطل (بطاقات أفاتار) → الموضوع (Chips) → الصوت (Chips) → شريط التكلفة التفصيلي (قصة ١٠ + أفاتار ١٠ + صوت ١٠ = ٣٠) → CTA "ابدأ التوليد".

### المكتبات
Header + كريدت → Tabs (مكتبتي / المكتبة العامة) → Grid بطاقتين بالصف. القصص الملكية تحمل pill "ملكي"، والمقفلة تحمل pill "١٠ كريدت".

### الإعدادات
Header → بطاقة الحساب (أفاتار + اسم + بريد + كريدت) → المحتوى الخاص (اصنع بطلك / اروِ بصوتك / مكتبتي) → الحساب والفوترة (شحن كريدت / المتجر / اللغة) → قانوني (خصوصية / شروط / خروج).

## 6) القواعد الصارمة المطبقة في التصميم
- الكريدت ظاهر في كل شاشة عدا السينما.
- كل زر موصول بوجهة حقيقية (لا Placeholder).
- شريط التكلفة يعرض التفصيل قبل الخصم (الخصم يتم بعد النجاح).
- القصص المحفوظة لا تُظهر زر "توليد" في المكتبة الخاصة.
- لا ربط منتج مادي بوعد كريدت داخل التطبيق.

## 7) مسار التنفيذ إلى Flutter
1. استبدال `lib/core/theme/app_colors.dart` بالقيم أعلاه.
2. إنشاء `lib/core/theme/app_typography.dart` و `app_spacing.dart`.
3. إنشاء ويدجتات مشتركة: `HikayatiAppBar`, `CreditPill`, `HeroBanner`, `QuickCard`, `StoryTile`, `HikayatiBottomNav`, `CTAButton`.
4. بناء الشاشات بنفس ترتيب HTML.

## 8) خطوط لاحقة
- نسخة Light (اختيارية لاحقاً) — حالياً Dark فقط كما تقتضي الهوية السينمائية.
- Motion: وهج يتحرك على hero ببطء، ستار parallax خفيف.
- Haptics: نقرة خفيفة على CTA الأساسي.
