## 1. ثبات الشخصية (Character Consistency)

- [ ] الشخصية الرئيسية تُطابق Character Bible في **جميع** المشاهد الستة
- [ ] نفس لون الشعر
- [ ] نفس لون العينين
- [ ] نفس ملامح الوجه (شكل، بشرة، مميزات)
- [ ] نفس الملابس الأساسية (لون، نمط، قلنسوة إن وُجدت)
- [ ] نفس الحذاء/الإكسسوارات
- [ ] لا تبدّل بين المشاهد (كأنها شخصية مختلفة)

**في حالة الفشل:** أعد توليد المشهد بـ prompt أقوى للـ Character Lock.

---

## 2. العمر صحيح

- [ ] الشخصية تبدو في عمرها المحدد (7 سنوات في Character Bible)
- [ ] **لا تظهر كطفل رضيع** (أصغر بكثير)
- [ ] **لا تظهر كمراهقة أو بالغة** (أكبر بكثير)
- [ ] الطول النسبي يناسب طفل (إذا كان معها بالغ، واضح الفرق)

---

## 3. اليدان

- [ ] 5 أصابع في كل يد (لا زيادة، لا نقص)
- [ ] الأصابع بأطوال طبيعية
- [ ] لا تشوّه في الراحة
- [ ] الأيدي بحجم متناسب مع الجسم

---

## 4. القدمان

- [ ] القدمان بأحجام طبيعية
- [ ] إذا كان يظهر الحذاء، واضح ومُطابق للـ Character Bible
- [ ] لا تشوّه

---

## 5. لا نصوص داخل الصورة

- [ ] لا نص عربي مكتوب في الخلفية
- [ ] لا نص إنجليزي
- [ ] لا أرقام
- [ ] لا لافتات أو إشارات عليها كتابة
- [ ] لا watermark
- [ ] لا signature
- [ ] لا logo

---

## 6. ثبات الملابس (Wardrobe Consistency)

- [ ] القطعة الرئيسية (مثل القلنسوة الحمراء) موجودة في **كل** مشهد
- [ ] الألوان متطابقة بين المشاهد
- [ ] لا تغيير مفاجئ في النمط
- [ ] لا تغيير في الحذاء
- [ ] إذا كان هناك إكسسوار (سلة، حقيبة)، ثابت في المشاهد المناسبة

---

## 7. جودة النص العربي

- [ ] لغة عربية فصحى سهلة
- [ ] لا أخطاء إملائية
- [ ] لا أخطاء نحوية
- [ ] الجمل قصيرة ومفهومة للأطفال
- [ ] لا خلط مع الإنجليزية
- [ ] لا ذكر "مشهد 1، مشهد 2" في النص
- [ ] تدفق طبيعي بين المشاهد

---

## 8. جودة القصة (Story Quality)

- [ ] البداية جذابة (لا تبدأ بكلام مُملّ)
- [ ] المشكلة واضحة
- [ ] التصاعد منطقي
- [ ] الحل مُقنع (ليس "فجأة تمت الحلول")
- [ ] النهاية مُرضية للطفل
- [ ] الدرس التربوي واضح بدون وعظ مباشر
- [ ] لا "ثقب" منطقي (شخصية تظهر ثم تختفي بلا سبب)

---

## 9. مناسبة للأطفال

- [ ] لا عنف قاسٍ
- [ ] لا رعب
- [ ] لا خوف مفاجئ مؤذٍ
- [ ] لا محتوى حزين مؤلم
- [ ] لا إشارات دينية مباشرة (القصة عامة)
- [ ] لا إشارات سياسية
- [ ] لا إعلانات أو علامات تجارية

---

## 10. المشاهد والأصوات

- [ ] 6 مشاهد بالضبط
- [ ] كل مشهد له صورة واحدة جاهزة (imageUrl)
- [ ] كل مشهد له ملف صوتي واحد (audio_url)
- [ ] كل ملف صوت يُطابق النص بدون أخطاء نطق واضحة
- [ ] مدة كل صوت مناسبة (ليس طويلاً جداً ولا قصيراً جداً)

---

## 11. جاهزية النقل إلى Supabase

- [ ] `public_stories` لديه الأعمدة: scenes_json, voice_type (تحقق بـ ALTER TABLE)
- [ ] JSON import جاهز في `06_import_ready/`
- [ ] الصور مرفوعة في Supabase Storage
- [ ] الصوتيات مرفوعة في Supabase Storage
- [ ] الروابط العامة (public URLs) تعمل
- [ ] summary مُنقّح ومكتوب
- [ ] category محدد
- [ ] price_credits = 10
