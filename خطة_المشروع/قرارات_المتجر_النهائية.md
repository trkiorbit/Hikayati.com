## Webhooks + Edge Functions (Phase 4)

### Flow الكامل

```
1. العميل في hikayati.com → يختار منتج → Cart → Checkout
2. POST إلى /api/checkout.php → يُنشئ Moyasar payment session
3. العميل يدفع → Moyasar ينجح
4. Moyasar webhook → Supabase Edge Function `payment-webhook`
5. Edge Function:
   a. يُنشئ row في orders (status=paid)
   b. إذا credit_card → يولّد كود 12 خانة → يُدرج في credit_cards
   c. إذا منتج مادي → يُرسل إلى Print partner API
   d. يُنشئ invoice PDF → يرفعه لـ Supabase Storage
   e. يُرسل بريد عبر Resend (order confirmation + PDF attachment)
6. العميل يستلم البريد
7. إذا credit_card → يفتح التطبيق → Redeem → الرصيد يزيد
8. إذا مادي → شريك الطباعة يُطبع → يشحن → status تحديث
```

### Supabase Tables

```sql
-- Orders
CREATE TABLE orders (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES auth.users(id) NULL,
  email TEXT NOT NULL,
  product_type TEXT,
  product_data JSONB,
  quantity INT DEFAULT 1,
  total_sar NUMERIC(10,2),
  status TEXT DEFAULT 'pending',
  payment_provider TEXT,
  payment_id TEXT,
  shipping_address JSONB,
  tracking_number TEXT,
  created_at TIMESTAMPTZ DEFAULT NOW(),
  updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- Invoices
CREATE TABLE invoices (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  order_id UUID REFERENCES orders(id) UNIQUE,
  invoice_number TEXT UNIQUE,
  pdf_url TEXT,
  total_sar NUMERIC(10,2),
  vat_sar NUMERIC(10,2),
  issued_at TIMESTAMPTZ DEFAULT NOW()
);

-- Credit Cards
CREATE TABLE credit_cards (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  code TEXT UNIQUE NOT NULL,
  credits INT NOT NULL,
  price_sar NUMERIC(10,2),
  status TEXT DEFAULT 'unused',
  order_id UUID REFERENCES orders(id),
  redeemed_by UUID REFERENCES auth.users(id),
  redeemed_at TIMESTAMPTZ,
  created_at TIMESTAMPTZ DEFAULT NOW()
);
```
