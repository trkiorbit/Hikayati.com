## 3. جداول Supabase الحالية

| الجدول | الغرض | الأعمدة الأساسية |
|---|---|---|
| `profiles` | بيانات المستخدم | user_id, credits, avatar_data, avatar_profile_summary, language |
| `stories` | قصص المستخدمين | id, user_id, title, scenes_json, cover_image, voice_type, created_at |
| `public_stories` | مكتبة عامة | id, title, scenes_json, cover_image, price_credits |
| `purchases` | سجل فتح القصص | user_id, story_id, unlock_type='access' |

**RPC Functions:**
- `deduct_credits(p_user_id, p_amount, p_reason)` — خصم آمن مع فحص رصيد
- `ensureProfileExists(userId)` — إنشاء ملف شخصي برصيد 100 عند الحاجة

**Storage Buckets:**
- `avatars/` — صور الأفاتار
- `story_audio/` — ملفات صوت القصص (`{userId}/{storyId}_scene_{index}.mp3`)

## 4. قواعد المعمارية الثابتة (لا تُكسر)

1. **الواجهة عرض فقط** — لا AI calls من داخل الشاشات مباشرة
2. **السينما عرض فقط** — لا API calls عند إعادة التشغيل، يقرأ من `audio_url` المحفوظ
3. **كل التوليد عبر Use Cases** — generate_story_use_case هو الوسيط الوحيد
4. **الأفاتار بالاستخراج فقط** — ممنوع إدخال العمر/الجنس/اللون يدويًا
5. **خصم الرصيد بعد النجاح فقط** — لا خصم إذا فشل أي خطوة
6. **مصدر واحد للحقيقة** — Supabase هو المرجع، لا fallback مُرمَّز

## 12. المهارات المطلوبة

### تقنية أساسية
- **Flutter/Dart** (متقدم) — شاشات، State Management، Animations، Platform Channels
- **Supabase** — PostgreSQL، RPC Functions، Storage، Edge Functions، Auth، Realtime
- **Riverpod** — موجود في المشروع، يحتاج توسيع للـ state management
- **GoRouter** — Navigation patterns، extra data passing

### APIs الخارجية
- **Pollinations AI** — Text Generation، Image Generation، TTS (نص + صور + صوت)
- **ElevenLabs** — Voice Cloning، Multilingual TTS V2
- **Google Vision API** — Face Detection، Feature Extraction للأفاتار
- **Google Sign-In** — OAuth flow

### إطلاق التطبيق
- **Apple App Store Connect** — Build submission، IAP، Review process، Certificates
- **Google Play Console** — AAB، Play Billing، Data Safety Form
- **Code Signing** — iOS certificates + provisioning profiles، Android keystore
- **Fastlane** (اختياري) — أتمتة البناء والرفع

### المتجر والمدفوعات
- **Apple StoreKit 2** — In-App Purchases في iOS
